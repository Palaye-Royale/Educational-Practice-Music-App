# Описание дипломного проекта: музыкальное веб-приложение

### Актуальность темы

Современные музыкальные стриминговые сервисы перегружены рекомендательными алгоритмами, рекламой и навязыванием «популярного» контента. Пользователю становится сложно организовать своё музыкальное пространство и управлять музыкой без отвлекающих элементов.

Таким образом появляется потребность в минималистичном инструменте, который предоставляет только базовые функции: поиск, добавление в избранное, создание плейлистов и т.д., и не перегружает пользователя лишней информацией и контентом.

Данный проект нацелен на создание веб-приложения, которое решает проблему избыточности популярных сервисов, предоставляя пользователю полный контроль над его музыкальной коллекцией.

### Целевая аудитория

* Любители музыки, которые хотят организовать свою коллекцию треков без навязывания "трендов". 
* Пользователи, уставшие от рекламы и визуального шума в интерфейсах крупных стримингов. 

### Предполагаемые функциональные возможности

* Регистрация и аутентификация пользователей: возможность создать аккаунт и войти в систему.
* Поиск музыки: интеграция с API Яндекс Музыки для поиска треков.
* Управление избранным: возможность добавлять треки в избранное и удалять их.
* Создание личной коллекции: сохранение треков в раздел "Мои треки".
* Плейлисты: создание, просмотр и редактирование собственных плейлистов, добавление треков в плейлисты и удаление из них.
* Интуитивный интерфейс: простой и чистый дизайн без отвлекающих элементов.

### Используемые технологии

* Backend: Node.js, фреймворк Fastify.
* База данных: SQLite (библиотека better-sqlite3).
* Frontend: HTML, CSS, JavaScript, шаблонизатор PUG.
* Интеграция с API: Неофициальное API Яндекс Музыки, HTTP-клиент.
* Хранение секретов: dotenv (переменные окружения).

### Ожидаемый результат

* Разработанное веб-приложение, доступное по локальному адресу.
* Пользователь может регистрироваться, искать музыку, создавать плейлисты и управлять своей коллекцией.
* Проект демонстрирует навыки работы с серверными технологиями, базами данных и внешними API.
