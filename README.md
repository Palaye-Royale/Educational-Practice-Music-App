# MyMusic

Веб-приложение для поиска музыки, создания плейлистов и управления коллекцией треков через Яндекс API.

## Запуск

```bash
# Установка зависимостей
npm install

# Запуск сервера
node server.js
```

Для поиска композиций необходимо получить токен Яндекс и создать файл .env в корне проекта:
YANDEX_TOKEN=твой_токен_яндекс_музыки

## Функционал
* Регистрация / вход 
* Поиск музыки через Яндекс API 
* Избранное, коллекция треков 
* Создание и управление плейлистами 

## Технологии
* Backend: Fastify, SQLite
* Frontend: PUG, Vanilla JS
* API: Яндекс Музыка

## Примечание
Воспроизведение nhtrf — заглушка. Для реального воспроизведения требуется доработка интеграции с API.
