# УП.02

## Музыкальный менеджер MyMusic

Веб-приложение для поиска музыки, создания плейлистов и управления коллекцией треков через Яндекс API.

### Запуск

```bash
# Установка зависимостей
npm install

# Запуск сервера
npm start
```

Для поиска композиций необходимо получить токен Яндекс.Музыка и создать файл .env в корне проекта:
YANDEX_TOKEN=ваш_токен_яндекс_музыки

### Функционал
* Регистрация и вход 
* Поиск музыки через Яндекс API 
* Управление избранным и коллекцией треков 
* Создание плейлистов

### Технологии
* Backend: Fastify, SQLite
* Frontend: PUG, Vanilla JS
* API: Яндекс Музыка

### Примечание
Воспроизведение треков — это заглушка. Для реального воспроизведения требуется доработка интеграции с API.


# УП.11

## База данных

Используется SQLite. Таблицы создаются автоматически при первом запуске:

| Таблица           | Описание                   |
|-----------------|--------------------------|
| users           | пользователи             |
| favorites       | избранные треки           |
| user_tracks     | коллекция «Мои треки»     |
| playlists       | плейлисты пользователей   |
| playlist_tracks | треки внутри плейлистов   |
